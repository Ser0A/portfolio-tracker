# Portfolio Tracker

Een persoonlijke portfolio tracker met live koersen, gebouwd met Python, Flask en SQLite.

## Features

- Live koersen via Yahoo Finance
- Winst/verlies berekening per asset
- Totaal overzicht
- Grafiek van je portfolio verdeling
- Donker thema via Tailwind CSS

## Installatie

1. Clone de repository
git clone https://github.com/Ser0A/portfolio-tracker.git

2. Installeer de dependencies
pip install flask yfinance

3. Start de app
python app.py

4. Open je browser op http://127.0.0.1:5000

## Technologie

- Python
- Flask
- SQLite
- Yahoo Finance (yfinance)
- Tailwind CSS
- Chart.js
